학습된 모델 파일(.pt)은 용량이 커서 GitHub에 포함되지 않습니다.

**다운로드 링크**: [Google Drive](https://drive.google.com/drive/folders/1Noow6HkhI6hkAuggptroiNmbUVGDbu1u?usp=sharing)

| 파일 | 크기 | 모델 | 역할 |
|------|------|------|------|
| `kcelectra.pt` | ~400MB | KcELECTRA-Base | 슬랭/욕설 전문가 |
| `soongsil.pt` | ~400MB | SoongsilBERT-Base | 안정적 베이스라인 |
| `roberta_base.pt` | ~400MB | KLUE-RoBERTa-Base | 고맥락 의미론 전문가 |
