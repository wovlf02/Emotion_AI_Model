## 2. 데이터셋 #1: K-MHaS

### 2.1 기본 정보

| 항목 | 내용 |
|------|------|
| **정식 명칭** | K-MHaS (Korean Multi-label Hate Speech) |
| **HuggingFace ID** | `jeanlee/kmhas_korean_hate_speech` |
| **출처 논문** | "K-MHaS: A Multi-label Hate Speech Detection Dataset in Korean Online News Comment" (COLING 2022) |
| **라이선스** | CC BY 4.0 |
| **수집 원천** | 한국 온라인 뉴스 댓글 |

### 2.2 데이터 구조

```
DatasetDict({
    train:      78,977건
    validation:  8,776건
    test:       21,939건
    ─────────────────────
    합계:      109,692건
})

Features:
  text:  Value('string')
  label: List(ClassLabel)
         → [0]=origin, [1]=physical, [2]=politics, [3]=profanity,
           [4]=age, [5]=gender, [6]=race, [7]=religion, [8]=not_hate_speech
```

### 2.3 샘플 예시

```python
{'text': '"자한당틀딱들.. 악플질 고만해라."', 'label': [2, 4]}
# label: [politics, age] → 정치 혐오 + 연령 혐오
```

### 2.4 다운로드 코드

```python

```

### 2.5 UnSmile 레이블 매핑

| K-MHaS label index | K-MHaS 이름 | → UnSmile 매핑 | 매핑 방식 |
|---|---|---|---|
| 0 | `origin` | `지역` | 직접 매핑 |
| 1 | `physical` | `기타 혐오` | 직접 매핑 |
| 2 | `politics` | `기타 혐오` | 직접 매핑 |
| 3 | `profanity` | `악플/욕설` | 직접 매핑 |
| 4 | `age` | `연령` | 직접 매핑 |
| 5 | `gender` | `여성/가족` 또는 `남성` | 텍스트 키워드 분석으로 분리 |
| 6 | `race` | `인종/국적` | 직접 매핑 |
| 7 | `religion` | `종교` | 직접 매핑 |
| 8 | `not_hate_speech` | 모두 0 (Clean) | 직접 매핑 |

**gender → 여성/남성 분리 전략**:
```python
female_keywords = ['김치녀', '맘충', '여자', '여성', '페미', '보슬', '한녀']
male_keywords = ['한남', '남자', '남성', '자지', '남충']
# 키워드 포함 시 해당 레이블, 미포함 시 양쪽 모두 1
```

|---|---|---|---|
| `기타 혐오` (F1: 54%) | 569건 | +수만 건 (physical+politics) | **구조적 해결** |

| 항목 | 내용 |
|------|------|
| **정식 명칭** | KOLD (Korean Offensive Language Dataset) |

| **수집 원천** | 네이버 뉴스 댓글 |

### 3.2 데이터 구조

```
DatasetDict({
    train: 40,429건  (단일 split)
})

Features:
  guid:        고유 ID
  source:      출처 ('naver_news')
  comment:     댓글 텍스트
  OFF:         공격성 여부 (bool)

```

```python
  'OFF': True,
  'TGT': 'group',
  'GRP': 'others-feminist',
  'raw_labels': [{
    'target_group': [['집단-성 정체성-여성']],

}
```

### 3.5 UnSmile 레이블 매핑

`raw_labels` → `target_group` 필드에서 매핑:

| KOLD target_group 패턴 | → UnSmile 매핑 |
|---|---|
| `집단-성 정체성-여성*`, `집단-성 정체성-페미니스트*` | `여성/가족` |
| `집단-성 정체성-남성*` | `남성` |
| `집단-성 정체성-성소수자*` | `성소수자` |
| `집단-국적/인종/민족-*` | `인종/국적` |
| `집단-연령-*` | `연령` |
| `집단-지역-*` | `지역` |
| `집단-종교-*` | `종교` |
| `OFF=True` + 위 매핑 해당 없음 | `기타 혐오` |
| `OFF=False` | 모두 0 (Clean) |

추가로 `OFF=True`인 모든 건에 욕설 키워드 검사를 수행하여 `악플/욕설` 레이블 부여.

```
DatasetDict({

Features:
```


### 4.4 UnSmile 레이블 매핑

| 조건 | → UnSmile 매핑 |
|------|----------------|
| `hate='hate'` 또는 `hate='offensive'` | `악플/욕설` = 1 |
| `contain_gender_bias=True` + 여성 키워드 | `여성/가족` = 1 |
| `contain_gender_bias=True` + 남성 키워드 | `남성` = 1 |
| `bias='others'` + `hate='hate'` | `기타 혐오` = 1 |
| `hate='none'` | 모두 0 |

나머지 카테고리(`성소수자`, `인종/국적`, `연령`, `지역`, `종교`)는 텍스트 키워드 보조 매핑 또는 0으로 처리.

### 5.4 UnSmile 레이블 매핑 — Pseudo-Labeling

APEACH는 이진 레이블(혐오 여부)만 있으므로 **직접 다중 레이블 매핑 불가**.

**전략: Phase 1 학습 완료 모델로 pseudo-labeling**

```
1. APEACH에서 class=1 (Spoiled) 샘플만 추출
2. Phase 1 앙상블 모델로 9차원 확률 예측
3. confidence ≥ 0.9 인 레이블만 채택
4. confidence < 0.9 인 레이블은 0으로 처리
5. 9차원 모두 0이 되는 샘플은 제외
```

| **규모** | 약 8,500건+ (대용량 parquet, 이미지 포함) |

### 7.2 데이터 구조

```
DatasetDict({
    train: 8,448건  (단일 split)
})

Features:
  instruct_text: Value('string')   ← 유해 텍스트 본문
```

| `/04.폭력/` | 512 | 6.1% | `기타 혐오` |
| `/05.범죄/` | 1,792 | 21.2% | (제외 — 혐오 표현과 무관) |
| `/06.허위정보및루머/` | 512 | 6.1% | (제외 — 혐오 표현과 무관) |

### 7.6 UnSmile 레이블 매핑

**활용 대상**: 비난혐오차별(2,560건), 욕설(768건), 폭력(512건) = **3,840건**  
**제외 대상**: 선정(768건), 범죄(1,792건), 허위정보(512건), 스팸광고(1,536건) = 4,608건

| 조건 | → UnSmile 매핑 |
|------|----------------|
| `src_path='비난혐오차별'` + `level2_type`에 '성별', '여성' 포함 | `여성/가족` |
| `src_path='비난혐오차별'` + `level2_type`에 '남성' 포함 | `남성` |
| `src_path='비난혐오차별'` + `level2_type`에 '인종', '외국인', '민족' 포함 | `인종/국적` |
| `src_path='비난혐오차별'` + `level2_type`에 '연령', '나이', '노인', '청소년' 포함 | `연령` |
| `src_path='비난혐오차별'` + `level2_type`에 '지역' 포함 | `지역` |
| `src_path='비난혐오차별'` + `level2_type`에 '종교' 포함 | `종교` |
| `src_path='비난혐오차별'` + `level2_type`에 '성소수자', '동성애' 포함 | `성소수자` |
| `src_path='비난혐오차별'` + 위 매핑 불가 | `기타 혐오` |
| `src_path='욕설'` | `악플/욕설` |
| `src_path='폭력'` | `기타 혐오` |

**추가 처리**: 모든 비난혐오차별 및 욕설 샘플에 대해 텍스트 기반 욕설 키워드 검사 → 해당 시 `악플/욕설` 추가 태깅.

### 7.7 기대 효과

| 현재 취약 클래스 | 추가 예상 건수 | 기대 효과 |
|---|---|---|
| `기타 혐오` (F1: 54%) | +1,500~2,000건 (비난혐오차별 중 기타 + 폭력) | 소수 클래스 보강 |
| `악플/욕설` (F1: 69%) | +768건 (욕설 카테고리) | 욕설 패턴 다양화 |
