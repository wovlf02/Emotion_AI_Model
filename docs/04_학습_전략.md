## 2. 손실 함수

### 2.1 BCEWithLogitsLoss

다중 라벨 분류를 위한 Binary Cross-Entropy 손실 함수를 사용합니다.

```python
# 손실 함수 정의
criterion = nn.BCEWithLogitsLoss(pos_weight=pos_weight)
```

**왜 BCEWithLogitsLoss인가?**

| 특성 | 설명 |
|------|------|
| Multi-Label | 각 라벨을 독립적인 이진 분류로 처리 |
| Numerical Stability | Sigmoid + BCE를 하나로 결합 (안정적) |
| pos_weight | 클래스 불균형 대응 가능 |

### 2.2 수식

```
L = -[y * log(σ(x)) + (1-y) * log(1-σ(x))]

where:
- x: 모델 출력 (logits)
- y: 타겟 레이블 (0 또는 1)
- σ: sigmoid 함수
```

### 2.3 클래스 가중치 (pos_weight)

소수 클래스에 더 높은 가중치를 부여합니다.

```python
def calculate_pos_weight(labels):
    """
    클래스별 positive weight 계산
    
    pos_weight = num_negative / num_positive
    """
    pos_counts = labels.sum(axis=0)
    neg_counts = len(labels) - pos_counts
    
    pos_weight = neg_counts / pos_counts
    
    # 과도한 가중치 방지 (최대 10배)
    pos_weight = torch.clamp(pos_weight, max=10.0)
    
    return pos_weight
```

| 클래스 | Positive 비율 | pos_weight |
|--------|---------------|------------|
| 악플/욕설 | 20.9% | 3.8 |
| 연령 | 4.0% | 10.0 (clamped) |
| 기타 혐오 | 3.7% | 10.0 (clamped) |

| 특성 | Adam | AdamW |
|------|------|-------|

### 5.3 Early Stopping

```python
class EarlyStopping:
    def __init__(self, patience=10, min_delta=0.001):
        self.patience = patience
        self.min_delta = min_delta
        self.counter = 0
        self.best_score = None
    
    def __call__(self, score):
        if self.best_score is None:
            self.best_score = score
        elif score < self.best_score + self.min_delta:
            self.counter += 1
            if self.counter >= self.patience:
                return True  # Stop training
        else:
            self.best_score = score
            self.counter = 0
        return False
```

| 파라미터 | 값 | 설명 |
|----------|-----|------|
| `patience` | 10 | 개선 없이 10 epoch 대기 |
| `min_delta` | 0.001 | 최소 개선 폭 |

## 7. 학습 루프

### 7.1 Train 코드

```python
def train_epoch(model, train_loader, optimizer, scheduler, criterion, device):
    model.train()
    total_loss = 0
    
    for batch in tqdm(train_loader):
        # 1. 데이터 GPU 이동
        input_ids = batch['input_ids'].to(device)
        attention_mask = batch['attention_mask'].to(device)
        labels = batch['labels'].to(device)
        
        # 2. Forward
        optimizer.zero_grad()
        logits = model(input_ids, attention_mask)
        
        # 3. Loss 계산
        loss = criterion(logits, labels)
        
        # 4. Backward
        loss.backward()
        
        # 5. Gradient Clipping
        torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
        
        # 6. Update
        optimizer.step()
        scheduler.step()
        
        total_loss += loss.item()
    
    return total_loss / len(train_loader)
```

### 7.2 Validation 코드

```python
def validate(model, val_loader, criterion, device):
    model.eval()
    total_loss = 0
    all_preds = []
    all_labels = []
    
    with torch.no_grad():
        for batch in val_loader:
            input_ids = batch['input_ids'].to(device)
            attention_mask = batch['attention_mask'].to(device)
            labels = batch['labels'].to(device)
            
            logits = model(input_ids, attention_mask)
            loss = criterion(logits, labels)
            
            probs = torch.sigmoid(logits)
            preds = (probs > 0.5).int()
            
            total_loss += loss.item()
            all_preds.append(preds.cpu())
            all_labels.append(labels.cpu())
    
    all_preds = torch.cat(all_preds)
    all_labels = torch.cat(all_labels)
    
    # 메트릭 계산
    f1_macro = f1_score(all_labels, all_preds, average='macro')
    
    return total_loss / len(val_loader), f1_macro
```

## 9. GPU 메모리 최적화

### 9.1 Mixed Precision (FP16/BF16)

```python
from torch.cuda.amp import autocast, GradScaler

scaler = GradScaler()

# Training loop
with autocast():
    logits = model(input_ids, attention_mask)
    loss = criterion(logits, labels)

scaler.scale(loss).backward()
scaler.step(optimizer)
scaler.update()
```

**효과**:
- 메모리 사용량 ~50% 감소
- 학습 속도 ~1.5배 향상

### 9.2 Gradient Accumulation

```python
accumulation_steps = 4
effective_batch_size = batch_size * accumulation_steps  # 16 * 4 = 64

for i, batch in enumerate(train_loader):
    loss = model(batch) / accumulation_steps
    loss.backward()
    
    if (i + 1) % accumulation_steps == 0:
        optimizer.step()
        optimizer.zero_grad()
```
